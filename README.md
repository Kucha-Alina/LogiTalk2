Спочатку треба завантажити в терміналі Auto-Py-to-Exe, пишемо pip install auto-py-to-exe. . 
Для відкривання цієї програми пишемо: auto-py-to-exe, програма відкриється автоматично.
Вкладаємо туди звичайний файл з нашим кодом, налаштовуємо exe файл та натискаємо конвертувати
Після завантаження цієї програми вона відкривається.
Після ціого наш exe файл відкривається.
Завантажуємо exe файл у GitHub пишемо Readme та завантажуємо у профіль.
Auto Py to Exe - це програма/бібліотека з графічним інтерфейсом, яка створює exe файл з нашим проєктом, яким зможуть користуватися всі.
